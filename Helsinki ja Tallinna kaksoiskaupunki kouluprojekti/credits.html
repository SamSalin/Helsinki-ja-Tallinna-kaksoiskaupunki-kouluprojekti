Tekijät

CSS Miami syntyi unelmasta. Unelmasta tehdä jotakin suurta. Valittettavasti suuruus ei tule itsestään vaan se tarvitsee kivenkovia tekijöitä. Onneksi nämä tekijät löysivät toisensa pääkaupunkiseudun melskeestä.


Mitä CSS Miami tarkoittaa? Se tarkoittaa irtiottoa arjesta. Se tarkoittaa itsensä likoon pistämistä joka päivä, joka tunti, joka sekunti. Se tarkoittaa sammumatonta tiedonnälkää. Se tarkoittaa voittamista.


Vakuutamme, että CSS Miami ei ole pyramidihuijaus* vaan jotain aivan uutta.

Pyramidihuijaus on laitonta rahankeräystä, joka toteutetaan ketjukirjeeseen tai verkostomarkkinointiin verrattavalla tavalla. Me emme ole niin kehittyneitä.
